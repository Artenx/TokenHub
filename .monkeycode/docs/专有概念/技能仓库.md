# 技能仓库

技能仓库是 TokenHub 的本地 Skill 元数据与文件管理领域。一个有效技能包以其根目录的 `SKILL.md` 为识别依据，后续导入服务将使用此规则校验上传和公开搜索结果。

## 结构

`SkillRepositoryConfig` 位于 `src/models.rs`，定义技能根目录与容量限制。`SkillRepositoryState` 保存公开来源、已导入技能和操作审计记录，并由 `AppState` 保存到配置目录的 `skill_repository.json`。

`SkillImportPreview` 用于管理员确认前的临时导入内容。预览只驻留内存，并在读取时移除已过期记录。

## 约束

- 有效技能包拥有唯一根目录 `SKILL.md`。
- 公开来源搜索采用匿名访问方式。
- 导入、替换和删除操作需要记录审计信息。
- 文件校验、导入和公开来源适配器属于后续实施任务。
