# 技能仓库

技能仓库是 TokenHub 的本地 Skill 元数据与文件管理领域。一个有效技能包以其根目录的 `SKILL.md` 为识别依据，本地导入服务已使用此规则校验 ZIP 归档和磁盘目录。

## 结构

`SkillRepositoryConfig` 位于 `src/models.rs`，定义技能根目录与容量限制。`SkillRepositoryState` 保存公开来源、已导入技能和操作审计记录，并由 `AppState` 保存到配置目录的 `skill_repository.json`。

`SkillImportPreview` 用于管理员确认前的临时导入内容。预览只驻留内存，并在读取时移除已过期记录。`skill_repository.rs` 会在导入时使用临时目录完成文件写入；替换操作先移动旧版本至备份目录，再提交新版本。

管理 API 会将已通过校验的包内容与预览 ID 关联保存。导入、替换或预览过期后，内存中的内容将被移除。远端预览只接收 HTTPS 归档地址，并限制地址属于已启用来源的域名；GitHub 来源还支持目录链接和 `SKILL.md` 链接，分支或标签用于定位仓库归档。

链接导入的公开 ZIP 下载会验证 HTTPS 默认端口、URL 凭据和域名解析结果。下载器只使用解析出的公开单播地址，关闭 HTTP 重定向，并在流式读取阶段执行总容量限制。

## 约束

- 有效技能包拥有唯一根目录 `SKILL.md`。
- 公开来源搜索采用匿名访问方式。
- 导入、替换和删除操作需要记录审计信息。
- 公开来源适配器支持 GitHub、SkillHub 与自定义 JSON 索引。
- 管理 API 已完成，页面属于后续实施任务。
