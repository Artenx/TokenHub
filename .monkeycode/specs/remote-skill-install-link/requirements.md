# 远程技能安装链接需求

## 简介

管理员可为本地技能创建包含高熵随机令牌的远程安装链接。客户端使用链接获取技能 ZIP 归档，并沿用本地技能包的有效性校验。管理员可配置链接有效期、复制链接、查看链接状态和提前撤销链接。

## 术语

- **安装链接**：指向单个本地技能、包含随机访问令牌且具有有效期的 HTTPS URL。
- **访问令牌**：由服务生成的高熵随机字符串，用于标识安装链接并授权归档下载。
- **链接记录**：持久化保存的安装链接元数据，包含技能目录、令牌摘要、创建时间、过期时间、撤销时间和使用统计。
- **安装客户端**：请求安装链接并将下载归档安装为技能的外部工具或用户环境。

## 已确认策略

- 有效期预设为 1 小时、24 小时、7 天和 30 天。
- 自定义有效期范围为 1 小时至 365 天。
- 管理员可选择持续有效或首次下载后失效的使用方式。
- 每个安装链接绑定创建时生成的技能内容快照。

## 需求

### 需求 1：创建安装链接

**用户故事：** 作为管理员，我希望为本地技能创建带有效期的安装链接，以便让外部安装客户端获取该技能。

#### 验收条件

1. 当管理员在本地技能详情中创建安装链接时，系统应当要求管理员选择有效期。
2. 当管理员提交有效期时，系统应当为该技能生成一个不可预测的访问令牌，并保存链接记录。
3. 当系统创建链接记录时，系统应当返回完整安装链接、创建时间和过期时间。
4. 当管理员选择预设时长时，系统应当使用界面展示的时长计算过期时间。
5. 当管理员输入自定义时长时，系统应当校验自定义时长位于产品定义的允许范围内。

### 需求 2：复制与管理安装链接

**用户故事：** 作为管理员，我希望复制、查看和撤销安装链接，以便控制技能分发范围。

#### 验收条件

1. 当管理员在安装链接列表中点击复制时，管理页面应当将完整安装链接写入剪贴板，并展示复制结果。
2. 当管理员查看本地技能详情时，系统应当展示该技能的链接状态、创建时间、过期时间、撤销时间和下载次数。
3. 当管理员撤销有效安装链接时，系统应当记录撤销时间，并立即停止该链接的下载授权。
4. 当管理员创建多个安装链接时，系统应当为每个链接维护独立的令牌、有效期、状态和下载计数。

### 需求 3：远程安装下载

**用户故事：** 作为安装客户端，我希望通过安装链接下载技能归档，以便在线安装指定技能。

#### 验收条件

1. 当安装客户端使用有效安装链接发起下载请求时，系统应当返回该技能的 ZIP 归档，并设置 ZIP 内容类型与下载文件名。
2. 当系统成功返回归档时，系统应当增加该链接的下载次数并记录最近下载时间。
3. 当访问令牌不存在、已过期或已撤销时，系统应当返回统一的访问失败响应，且响应内容不披露技能目录和链接状态。
4. 当安装链接关联的技能已被删除时，系统应当返回统一的访问失败响应。
5. 当安装链接关联的技能内容发生更新时，系统应当按产品定义的内容版本策略提供归档。

### 需求 4：令牌与状态持久化

**用户故事：** 作为管理员，我希望安装链接在服务重启后继续保持状态，以便链接生命周期稳定可追踪。

#### 验收条件

1. 当系统保存链接记录时，系统应当仅持久化访问令牌摘要，运行期使用摘要匹配访问令牌。
2. 当服务启动时，系统应当恢复未过期和已撤销链接记录的管理状态。
3. 当链接达到过期时间时，系统应当将链接视为过期，并停止归档下载授权。
4. 当系统处理过期链接记录时，系统应当保留链接记录供管理员查看和审计。

### 需求 5：安全与审计

**用户故事：** 作为管理员，我希望安装链接的访问和管理动作可追溯，以便排查技能分发情况。

#### 验收条件

1. 当管理员创建或撤销安装链接时，系统应当记录审计事件、操作时间、关联技能和链接标识。
2. 当安装客户端下载技能时，系统应当记录链接标识、下载时间和下载结果。
3. 当系统生成安装链接时，系统应当通过加密安全随机源生成访问令牌。
4. 当管理员接口接收创建、查询或撤销安装链接请求时，系统应当执行现有管理员会话鉴权。
