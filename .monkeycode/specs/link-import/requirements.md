# 链接导入需求

## 简介

管理员可在技能仓库中粘贴公开链接，预览并确认导入符合本地技能包规则的技能。

## 术语

- **GitHub 技能链接**：指向公开 GitHub 仓库目录或 `SKILL.md` 文件的 HTTPS 链接。
- **公开 ZIP 链接**：指向技能 ZIP 归档的 HTTPS 链接。
- **导入预览**：服务完成下载和校验后生成的、等待管理员确认的临时技能包。

## 需求

### 需求 1：链接导入入口

**用户故事：** 作为管理员，我希望粘贴技能链接，以便直接导入公开技能。

#### 验收条件

1. 当管理员在技能仓库页面提交链接时，系统应当发送链接预览请求。
2. 当链接指向有效技能包时，系统应当展示目录名、文件清单、冲突状态和预览过期时间。
3. 当管理员确认预览时，系统应当沿用现有导入确认和替换流程。

### 需求 2：GitHub 链接解析

**用户故事：** 作为管理员，我希望导入 GitHub 上某个目录中的技能，以便使用仓库中的独立技能包。

#### 验收条件

1. 当管理员提交 GitHub 目录链接时，系统应当下载对应分支或标签的仓库归档，并提取目标目录。
2. 当管理员提交 GitHub `SKILL.md` 链接时，系统应当将父目录作为目标技能目录。
3. 当目标目录包含 `SKILL.md` 时，系统应当交由现有 ZIP 校验流程生成导入预览。

### 需求 3：公开 ZIP 链接处理

**用户故事：** 作为管理员，我希望导入公开 HTTPS ZIP 链接，以便使用不在预置来源中的技能包。

#### 验收条件

1. 当管理员提交公开 HTTPS ZIP 链接时，系统应当下载归档并交由现有 ZIP 校验流程处理。
2. 当 URL 包含凭据、缺少主机、使用非 HTTPS 协议或使用受限端口时，系统应当返回明确错误。
3. 当解析到的主机地址属于回环、私有、链路本地、未指定或多播地址时，系统应当返回明确错误。
4. 当远端响应重定向时，系统应当返回明确错误。
5. 当下载内容超过技能仓库的总容量限制时，系统应当返回明确错误。

### 需求 4：可诊断错误

**用户故事：** 作为管理员，我希望看到链接预览失败的原因，以便修正链接或选择有效来源。

#### 验收条件

1. 当 GitHub 链接无法定位分支、标签或技能目录时，系统应当返回链接格式说明。
2. 当远端下载返回非成功状态时，系统应当返回 HTTP 状态码。
3. 当归档结构不符合技能包规则时，系统应当返回现有校验产生的错误信息。
